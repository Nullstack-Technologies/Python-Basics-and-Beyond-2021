##  Usage

Make sure you have python 3.0 and virtualenv instaled

Make Virtual Enviroment:

```sh
virtualenv venv
```


Activate Virtual Enviroment:

```sh
source venv/bin/activate
```

Install Packages:

```sh
pip install -r requirements.txt
```

Running any code:

```sh
python <file_name>
```
